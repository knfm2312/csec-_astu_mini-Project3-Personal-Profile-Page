# Toggle Text (ON / OFF)

## Overview
A simple web project built using **HTML**, **CSS**, and **JavaScript** that toggles text between **ON** and **OFF** when a button is clicked.  
It demonstrates basic **DOM manipulation**, **event handling**, and **responsive layout design** without using any frameworks.

---

## Features
- Displays “OFF” by default and changes to “ON” on button click.  
- Fully responsive and centered layout using **Flexbox**.  
- Clean design built with **vanilla JavaScript** and **CSS**.  

---

## Technologies Used
- **HTML5** – Structure  
- **CSS3** – Styling and layout  
- **JavaScript (ES6)** – Interactivity
